# College Student Database (SQL)

## Overview
This project demonstrates basic SQL operations using a simple **Student table**. It includes database creation, table structure, and examples of retrieving and analyzing student data.

## Table Structure
The database contains a **Student** table with the following fields:
- Roll Number
- Name
- Marks
- Grade
- City

## Concepts Covered
- Data retrieval
- Filtering records using conditions
- Sorting data
- Limiting results
- Aggregate functions (COUNT, SUM, AVG, MAX, MIN)
- Grouping data (e.g., counting students in each city)

## Learning Outcome
This project helps in understanding fundamental SQL concepts used for managing and analyzing data in relational databases.
